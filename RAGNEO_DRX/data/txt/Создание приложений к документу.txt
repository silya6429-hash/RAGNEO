Создание приложений к документу Приложение - документ, который уточняет или подробно раскрывает содержание основного документа. Приложения создаются для входящих, внутренних и исходящих документов. Приложения можно добавить несколькими способами в зависимости от их количества: - создание одного приложения; 1. В карточке документа на панели действий в выпадающем списке Связи выберите пункт Создать приложение. 2. Заполните поля в открывшейся карточке: *Вид документа, например, приложение к документу. *Ведущий док. Документ, для которого создается приложение, например, приказ или исходящее письмо. Цепочка связанных документов отобразится на закладке «Связи». *Содержание. Краткий заголовок приложения. 3. Добавьте содержимое приложения с помощью кнопки Создать из шаблона, или Создать со сканера на панели действий. Создать из файла, 4. Настройте права доступа на документ. По умолчанию для приложения копируются права доступа из ведущего документа. 5. Сохраните карточку документа. 6. При необходимости отредактируйте содержимое документа. - загрузка нескольких приложений. 1. В карточке документа на панели действий в выпадающем списке Связи выберите пункт Добавить приложения: 2. В открывшемся окне по кнопке «...» выберите файлы для загрузки: Стоит учитывать некоторые особенности загрузки: - имена файлов должны быть уникальными; - максимальное количество файлов для загрузки – 10, допустимый размер файла – до 25 МБ. 3. Нажмите на кнопку Загрузить. Загружаемые файлы создаются с типом «Приложение к документу». Карточки приложений автоматически заполняются и привязываются к ведущему документу. Посмотреть все связанные с документом приложения можно в его карточке на закладке «Связи»: Если необходимо добавить больше десяти приложений, нажмите на кнопку Добавить приложения повторно. 4. Настройте права доступа на приложения. По умолчанию для них копируются права доступа из ведущего документа. 5. Сохраните карточку документа.
